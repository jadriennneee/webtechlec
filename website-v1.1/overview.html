Overview of the World Wide Web

What is the World Wide Web?

  • The World Wide Web is a collection of Web resources and applications that serves as a tool for users to share and obtain information through the Internet.It is typically known as a representation of human knowledge around the globe.
  • Its Web resources are Hypertext A software system used to link one text-based resource to another. and Hypermedia An extended feature of Hypertext which enables handling of multimedia resources such as images and videos. based which enables resources to be linked to other resources.
    • The terms Hypermedia and Hypertext were invented by Ted Nelson
  • Applications are software tools used by users to access the Web resources.
    • Two forms of Application
      • Web Server
        • It hosts/houses electronic resources.
        • It gives response to clients’ request.
      • Web Client
    • “Spider”- A Web crawler application which performs Web Indexing to obtain all of the information for a certain topic in the Web.

Internet

  • A common misconception is that the Web is the same thing as the Internet.
  • It serves as a medium between the Web and the Web browser such that it allows users to access information with the use of the Web browser.
  • It is a collection of computers connected to a network.
    • Composed of the following:
      • Hosts
      • Media
      • Protocol

Web Browser

  • A typical tool used to interact with the Web.
    • Categories
      • Static - Information displayed in the website does not change or alter.
      • Dynamic - Generates or displays information that varies and depends on the client.

History of the World Wide Web

  • 1989
    • The World Wide Web was invented by Sir Tim Berners-Lee Father of modern Web in 1989 with the main purpose of allowing scientists to easily share scientific resources or documents to other researchers around the world.
    • The World Wide Web was developed at CERN, a company Sir Tim Berners-Lee works for.
    • Sir Tim Berners-Lee proposed the three Web Technologies:
      • HTTP
      • HTML
      • URL
  • 1990
    • CERN hosted the first website developed by Sir Tim Berners-Lee in Switzerland.
  • 1993
    • CERN made the World Wide Web open source such that users around the world can alter it and improve it without a cost.
  • 1994
    • The Web Consortium (W3C) An international organization that provides standards in developing the Web was founded by Sir Tim Berners-Lee.
