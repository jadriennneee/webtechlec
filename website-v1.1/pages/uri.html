Uniform Resource Identifier (URI)

What is URI?

  • It is a sequence of characters that provides a unique address to identify the path/location of a certain HTTP Resource.

Syntax of URI

  • Scheme - It indicates the protocol used in accessing the web resource.
  • Authority
    • (Deprecated) It contains the user information/authentication credentials
    • Host - It is associated with a domain name.
      • Domain Name - It is resolved into an IP Address with the use of Domain Name System (DNS).
    • Port Number (optional) - It indicates the port of connection.
  • Path
    • Indicates the path to the location of the resource.
    • It may refer to a static or dynamic resource.
  • Query
    • A question mark (?) indicates the start of a query issued by the client.
    • It serves as an additional parameter in the URI.
    • It is typically provided as a key=value pair.
    • It may be URL-Encoded It is necessary so that servers can understand the characters used in the URI. .
  • Fragment Identifier (optional) - Identifies a specific part of the resource.
