Hypertext Markup Language (HTML)

What is HTML?

  • It is a language used to mark-up documents in the World Wide Web which is initially used for structure and content but in the modern Web, HTML is solely used for semantics and content.
  • It allows users to obtain and share resources in different forms such as video, graphics, documents, etc.

HTML Standards

  • HTML 1.0
    • It was the first release of HTML.
  • HTML 2.0 (RFC 1866)
    • Became the standard for website design and added core features to HTML.
    • It became standardized as a Request for Comments (RFC).
  • HTML 3.2 (W3C Recommendation)
    • Provided a set of extension to the previous version of HTML.
    • W3C has been founded such that it provides standards in developing the Web with the use of HTML.
    • Became the official standard for the developers.
  • HTML 4.01 (W3C Recommendation)
    • It provided a richer interface.
    • Introduced the proper usage of HTML making it solely for semantics and content and opted for stylesheet in structuring the website.
    • Introduced deprecation of presentational elements.
    • Versions
      • Strict - It forbids the use of eprecated elements.
      • Transitional - It allows deprecated elements for backward compatibility.
  • XHTML
    • It is strictly formatted as XML.
      • XML - An easily parsed structure that allows users to create their own tags.
    • XML defined rules and ways on how HTML 4.01 tags and attributes are used.
    • XHTML vs. HTML 4.0
      • XHTML supports nested elements.
      • XHTML attribute names and elements must be in lower case format.
      • End tags are required in XHTML.
  • HTML5
    • The first draft was edited by Ian Hickson (2008).
    • W3C Recommendations:
      • October 2014 (HTML5)
      • November 2016 (HTML 5.1)
      • December 2017 (HTML 5.2)
      • HTML Living Standard (WHATWG)
    • Became the specification the developers must follow in developing for the Web and provided additional elements, attributes and tags that can be used by developers.
      • Semantic Tags Added:
        • Header
        • Main
        • Nav
        • Article
        • Section
        • Aside
        • Address
        • Footer

HTML Document Basic Structure

  • Version Information (i.e DOCTYPE)
  • Basic Components
    • Document Type Declaration (DOCTYPE)
      • HTML 4.01
      • XHTML
      • HTML5
    • Elements
      • Tags
        • Content Models:
          • Flow
          • Metadata
          • Sectioning
          • Phrasing
          • Embedded
          • Interactive
          • Heading
        • Void Elements
      • Attributes and Attribute Values
        • Global Attributes - Attributes can be used by any HTML element.
        • Custom Non-visible Data Attributes - It can be used in creating own attributes which is then referenced to a script.
        • WAI-ARIA attributes - It is intended for providing accessibility.
        • Element-specific Attributes - Elements used if attributes are not global.
      • MathML, SVG
    • Character References
      • It provide ways by which developers can display something literally.
      • It is commonly used as an escape sequence.
