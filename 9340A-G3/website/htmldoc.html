Hypertext Markup Language (HTML)

What is HTML?

  • It is a language used to mark-up documents in the World Wide Web which is initially used for structure and content but in modern Web, HTML is solely used for semantics and content. It allows users to obtain and share resources in different forms such as video, graphics, documents, etc.

HTML Standards

  • HTML 1.0
    • It was the first release of HTML
  • HTML 2.0 (RFC 1866)
    • Became the standard for website design and added core features to HTML
    • It became standardized as an RFC
  • HTML 3.2 (W3C Recommendation)
    • Provided set of extensions to the previous version of HTML
    • W3C has been founded such that it provides standards in developing the Web with the use of HTML
    • Became the official standard for the developers.
  • HTML 4.01 (W3C Recommendation)
    • Provided a richer interface
    • Introduced the proper usage of HTML such that it is solely used for semantics and content and opted for stylesheet in structuring the website.
    • Introduced deprecation of presentational elements
    • Versions
      • Strict - forbids the use of deprecated elements
      • Transitional - allows deprecated elements for backward compatibility
  • XHTML
    • It is strictly formatted as XML
      • XML - easily parsed structure that allows users to create their own tags
    • XML defined rules and ways on how to HTML 4.01 tags and attributes are used
    • XHTML vs. HTML 4.0
      • XHTML supports nested elements
      • XHTML attribute names and elements must be in lower case format
      • End tags are required in XHTML
  • HTML5
    • The first draft was edited by Ian Hickson (2008)
    • W3C Recommendations:
      • October 2014 (HTML5)
      • November 2016 (HTML 5.1)
      • December 2017 (HTML 5.2)
      • HTML Living Standard (WHATWG)
    • Became the specification that developers must follow in developing for the Web and provided additional elements, attributes and tags that can be used by developers.
      • Semantic Tags Added:
        • Header
        • Main
        • Nav
        • Article
        • Section
        • Aside
        • Address
        • Footer

        • Global Attributes - Attributes can be used by any HTML element

        • Element-specific Attributes - If attributes are not global

    • Character References
      • Provide ways by which developers display something literally
