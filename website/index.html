Why Learn Web Technologies?

Web tech is a way where computer can communicate to one another by using markup languages and multimedia packages. It involves the use of hypertext markup language and cascading stylesheet. Learning web technologies is crucial in creating a standard and well optimized website for it teaches the fundamentals of web protocol as well as security
